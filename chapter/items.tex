\section{项目说明}

本项目已经在GitHub开源,使用\textbf{Git}进行管理,创建的初衷仅为个人进行期末考试的课程复习使用,也欢迎其他看到本项目的同学使用。由于作者水平有限，本项目比较屎山，还请见谅。

本项目远程仓库地址为\url{https://github.com/nsyhfxc/Lisan-math}.如需克隆本项目,您可以使用以下命令:

\begin{lstlisting}[language=bash]
	git clone https://github.com/nsyhfxc/Lisan-math.git
\end{lstlisting}

本项目暂不接受其他人的\texttt{commit}修改，提出修改意见请使用GitHub的\texttt{issue}

\subsection{参考来源与整理说明}

本文档后续章节参考并重组了 CSDN 博客《离散数学复习笔记（已完结）》中的知识框架，原文地址为：
\url{https://blog.csdn.net/qq_51594676/article/details/122138122}。

原文包含较多以图片呈现的公式、表格和图示。本笔记已将这些图片中的主要内容识别并改写为 \LaTeX{} 公式、表格和文字说明，同时结合本项目原有模板补充了例题与常用结论，便于期末复习。
